# ToyLink Direct

ToyLink Direct is a simple web platform that connects toy sellers directly with customers.

## Features
- Browse toys easily
- Simple and clean UI
- Fast product loading
- Responsive design

## Tech Stack
- React
- Vite
- Tailwind CSS

## Installation

git clone https://github.com/Devimanoj2005/toylink-direct.git
cd toylink-direct
npm install
npm run dev
